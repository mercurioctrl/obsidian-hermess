Actualmente la categoría **Camaras** esta mezclando muchas categorias de cámaras digitales, cámaras ip y webcams, la idea es poder separar esta mezcla donde quede algo asi:

```
Imagen / Fotografía
└── Cámaras Digitales
    ├── Cámaras compactas
    ├── Cámaras deportivas
    └── Accesorios de cámara

Hogar / Seguridad
└── Cámaras IP
    ├── Cámaras de vigilancia
    ├── Cámaras WiFi
    ├── Cámaras de seguridad
    └── Kits de seguridad

Componentes / Periféricos
└── Webcams
    ├── Webcams USB
    └── Cámaras para PC
```

Con esta query se ven productos que están en **Cámaras Digitales** pero son de la categoría **Cámaras IP / Seguridad**:

```
SELECT
    p.id,
    p.titulo,
    c.nombre AS categoria_actual
FROM CS.dbo.productos p
INNER JOIN LO.dbo.categorias c
    ON c.id = p.id_categoria_lo
WHERE p.id_categoria_lo = 314
AND (
       p.titulo LIKE '%IP%'
    OR p.titulo LIKE '%WIFI%'
    OR p.titulo LIKE '%WI-FI%'
    OR p.titulo LIKE '%HIKVISION%'
    OR p.titulo LIKE '%DAHUA%'
    OR p.titulo LIKE '%EZVIZ%'
    OR p.titulo LIKE '%VIGILANCIA%'
    OR p.titulo LIKE '%SEGURIDAD%'
    OR p.titulo LIKE '%DOMO%'
    OR p.titulo LIKE '%DVR%'
    OR p.titulo LIKE '%NVR%'
)
ORDER BY p.titulo;
```

Con esta query se ven productos que están en **Cámaras IP** pero son accesorios/soportes y no cámaras:

```
SELECT
    p.id,
    p.titulo,
    c.nombre AS categoria_actual
FROM CS.dbo.productos p
INNER JOIN LO.dbo.categorias c
    ON c.id = p.id_categoria_lo
WHERE p.id_categoria_lo = 203
AND (
       p.titulo LIKE '%SOPORTE%'
    OR p.titulo LIKE '%BASE%'
    OR p.titulo LIKE '%MOUNT%'
    OR p.titulo LIKE '%BRAZO%'
    OR p.titulo LIKE '%ACCESORIO%'
)
ORDER BY p.titulo;
```
