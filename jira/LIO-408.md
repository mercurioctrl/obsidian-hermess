## Descripción

### 🎯 Esquema de referidos propuesto para la plataforma

El sistema de referidos busca incentivar a que los usuarios actuales inviten a nuevos compradores a través de un **link o código único**. Cuando alguien se registra o compra usando ese código, ambos reciben un beneficio.

#### 🧩 ¿Cómo funciona?

1. **Cada usuario registrado obtiene un código o link de referido único.**  
Ejemplo: `libreopcion.com.ar?producto_35434&refer=lucas123`
2. **El usuario comparte ese link con sus amigos, redes o conocidos.**
3. **Cuando alguien entra con ese link y realiza una compra, se activan los beneficios:**  - El nuevo comprador (referido) obtiene un incentivo (ej: descuento en su primera compra).
  - El que lo invitó (referidor) gana un premio (ej: crédito, puntos, saldo, etc.).
4. **Todo queda registrado en el sistema** para poder:  - Validar compras reales
  - Prevenir fraude
  - Mostrar un resumen al usuario (cuántos referidos tiene y qué ganó)

#### 🔐 Controles

- El sistema detecta si se trata de un abuso (mismas IPs, cuentas falsas, bots, etc.).
- Las recompensas se otorgan solo cuando el referido hace una compra válida.

---

### 📌 Objetivo

Transformar a nuestros propios usuarios en promotores del sitio, bajando el costo de adquisición y generando crecimiento orgánico con beneficios medibles para ambas partes.
