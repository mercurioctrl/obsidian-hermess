# saMiniSiteCodes

Micrositio de canje de códigos de activación para productos ASUS / ROG (Resident Evil).
Formulario público para solicitar código, panel de admin para gestionar claims y enviar códigos por email.

**Estado:** Promoción finalizada (2026-04-15). Sitio activo solo para instrucciones y soporte.

**Stack:** PHP 8.2 · Nginx · MySQL 8 · Docker Compose
**Repo:** `asus-saMiniSiteCodes`
**Puerto local:** `http://localhost:8472`
