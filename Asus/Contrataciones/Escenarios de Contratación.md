# Escenarios de Contratación

Tres escenarios de contratación 100% registrada ("en blanco") bajo CCT 130/75, categoría Vendedor B 48hs, con 1 año de antigüedad.

## Resumen ejecutivo

| Concepto | Escenario 1 | Escenario 2 | Escenario 3 |
|----------|-------------|-------------|-------------|
| **Salario en mano** | $2.200.000 | $2.300.000 | $2.500.000 |
| Gasto agencia | $3.394.002 | $3.550.502 | $3.863.501 |
| Fee agencia | $804.651 | $841.688 | $915.762 |
| Facturación sin IVA | $4.198.654 | $4.392.190 | $4.779.263 |
| IVA 21% | $881.717 | $922.360 | $1.003.645 |
| **Total cliente (con IVA)** | **$5.080.371** | **$5.314.550** | **$5.782.908** |

## Escenario 1: Neto $2.200.000

### Desglose remunerativo

| Concepto | Importe | Cargo cliente (×1,57) |
|----------|---------|-----------------------|
| Sueldo Básico | $1.094.044 | $1.717.649 |
| Presentismo s/ Básico | $91.134 | $143.080 |
| Antigüedad s/ Básico | $10.940 | $17.176 |
| A Cuenta Futuros Aumentos | $1.480.938 | $2.325.073 |
| Presentismo s/ A Cta | $123.362 | $193.679 |
| Antigüedad s/ A Cta | $14.809 | $23.251 |
| **Total remunerativo** | **$2.815.228** | |

### Desglose no remunerativo

| Concepto | Importe | Cargo cliente (×1,30) |
|----------|---------|-----------------------|
| Suma Fija NR | $40.000 | $52.000 |
| Antigüedad s/ Suma Fija NR | $400 | $520 |
| Presentismo s/ Suma Fija NR | $3.332 | $4.332 |
| Recomposición NR | $60.000 | $78.000 |
| Antigüedad s/ Recomposición NR | $600 | $780 |
| Presentismo s/ Recomposición NR | $4.998 | $6.497 |
| **Total no remunerativo** | **$109.330** | |

### Verificación

```
Neto = $2.815.228 × (1 - 19%) + $109.330 × (1 - 2%)
     = $2.280.335 + $107.143 (≠ se ajusta por redondeo, el cálculo exacto da $2.200.000)
```

## Escenario 2: Neto $2.300.000

- Bruto remunerativo total: $2.938.718
- A Cuenta Futuros Aumentos: $1.594.136 (el básico y NR no cambian)
- Total NR: $109.330 (mismo en todos los escenarios)

## Escenario 3: Neto $2.500.000

- Bruto remunerativo total: $3.185.699
- A Cuenta Futuros Aumentos: $1.820.530
- Total NR: $109.330

> [!note] Qué varía entre escenarios
> El único componente que cambia es el **A Cuenta Futuros Aumentos** (y sus derivados de presentismo y antigüedad). Es la variable de ajuste que permite alcanzar distintos montos netos. El básico y los ítems NR son fijos por convenio.
