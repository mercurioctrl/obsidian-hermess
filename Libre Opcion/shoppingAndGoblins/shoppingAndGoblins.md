# shoppingAndGoblins — LIBRE RUN

Mini-juego *endless runner* (Grinch/Goblin) para [[Libre Opcion]].
Base de conocimiento del proyecto.

**Repo:** `git@github.com:BluIncStudio/miniJuegos-shoppingAndGoblins.git`
**Working dir:** `/var/www/purchasesAndGoblins`
**Stack:** HTML5 Canvas + Web Audio + localStorage (archivo único, sin dependencias).

## Qué es

Runner infinito estilo el dino de Chrome: el personaje corre con un carrito, salta
paquetes y drones, y junta productos de computación (10–150 pts). Ranking del día en
localStorage + panel de organizador para configurar el premio.
