# 03 · Fase 2 — Rentabilización y Profundidad

> [!info] Periodo: Meses 6–18 · Crecimiento rentable
> PDF: [[adjuntos/LibreOpcion-Fase2-Rentabilizacion-y-Profundidad.pdf|📄 Fase 2 (PDF)]]

## 🎯 Objetivo
Multiplicar los ingresos por usuario y por vendedor **sin depender solo de la comisión**, profundizando el nicho de tecnología para subir ticket y recompra.

## Prioridades estratégicas

1. **Retail media** — fabricantes pagan visibilidad. Alto margen, sin inventario.
2. **Financiación como producto** — acuerdos preferentes, planes exclusivos.
3. **Venta propia (1P) selectiva** en categorías estrella (componentes, gaming).
4. **Servicios de valor** — montaje, garantía extendida, instalación.
5. **Comunidad y contenido a escala** — reviews, esports, creadores.
6. **Línea B2B** — pymes/empresas con factura y volumen.

## Stack de monetización

| Capa | Margen / esfuerzo |
|---|---|
| Comisión marketplace | Base |
| Retail media | Alto / medio |
| Financiación | Alto / medio |
| Servicios | Medio |
| 1P selectivo | Medio / alto esfuerzo |
| B2B | Medio |

> [!tip] Principio rector
> La rentabilidad no viene de subir la comisión, sino de **apilar capas** donde cada una monetiza algo distinto.

## KPIs
Ingreso por capa · ticket promedio · recompra · CAC vs LTV · ingresos B2B.
