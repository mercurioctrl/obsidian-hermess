# PEGA

Base de conocimiento del proyecto PEGA — plataforma de comparación de precios de hardware gamer.

## Descripción

Plataforma multi-componente para comparar precios de hardware gamer entre múltiples revendedores.

- **Backend**: Laravel 10 REST API (PHP 8.1+, SQL Server, Redis)
- **Frontend**: Nuxt 2 / Vue 2 (Ant Design Vue, Chart.js)
- **Scraper**: FastAPI Python (agrega precios de HardGamers, CompraGamer, MercadoLibre, TiendaMía)
