## Descripción

Hola Catri! 

El pixel de meta tira un error y para solucionarlo me pide la siguiente info: 

Cómo establecer el valor (precio)

**En píxel de Meta:**

- Abre el código de tu sitio web.


- Busca fbq('track', 'Purchase') para localizar el evento de compra.


- Actualiza el campo **value**.



Información del campo **value**:

- Debe ser un valor numérico superior a 0.


- Puede incluir el punto decimal. Por ejemplo, 9.99 o 9.



**Si se establece una conexión a través de una app de socio:**Busca y actualiza el valor en la configuración de la app o ponte en contacto con el socio para obtener ayuda.
