## Descripción

1. Realizaremos una refactorización en el recurso para normalizar el CUIT/CUIL, eliminando los caracteres al agregar o editar CUIT/CUIL. Esto facilitará la identificación cuando se realicen búsquedas.

"20-32376224-5" -> "20323762245"

```
{API_URL}/v1/clients
```



2. Diseñaremos una consulta SQL que se encargue de purificar los campos de CUIT/CUIL, removiendo todos aquellos caracteres que no sean dígitos. Este proceso garantizará que en la base de datos solo permanezcan los valores numéricos esenciales.

[attachment]
```
UPDATE NewBytes_DBF.dbo.clientes
SET cdnicif = REPLACE(cdnicif, '-', '')
```
