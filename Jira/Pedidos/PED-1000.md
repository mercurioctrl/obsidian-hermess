## Descripción

Agregaremos al modal de ordenes las opciones enecesarias para gestionar las intervenciones técnicas antes de enviar un pedido. Esto incluye la configuración de indicadores para acciones como “armado de PC”, “actualizar la BIOS” e “instalar el sistema operativo”.

**Ejemplo del segmento:**

[attachment]

Para eso usaremos los recursos de [link](https://bluinc.atlassian.net/browse/PED-999) 

```
GET {API_URL}/v1/orders/{branch}-{order}/technicalIntervention
```

```
{
  "orderNumber": "10405377",
  "branchNumber": "0002",
  "assemblePc": true,
  "updateBios": false,
  "installOS": true
}

```

```
PATCH /v1/orders/{branch}-{order}/technicalIntervention
```

```
{
  "assemblePc": true,
  "updateBios": false,
  "installOS": true
}
```
