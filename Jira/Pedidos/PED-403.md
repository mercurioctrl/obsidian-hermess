## Descripción

API - Refactor - Listar ordenes - Filtro "reservados" debe ser sodo los que tienen pedido (cestado = s), pero no estan liquidados.



Es decir que el filtro debe esar combinado con algo como `MS_REMITOS_CABECERA.REMITO_FP IS NULL `o similar, para poder ver solo los que están en ese estado, pero no liquidados.
