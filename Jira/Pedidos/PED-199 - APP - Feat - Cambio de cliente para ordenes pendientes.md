## Descripcion

Al hacer click derecho sobre una orden, se debe agregar la opción “Cambiar cliente”. Solo cuando el pedido esta pendiente podremos usar esta opción.

La misma desplegara un modal donde podemos seleccionar el cliente de destino según el repositorio clientes y luego confirmar el movimiento. 

[link](https://lioteam.atlassian.net/browse/PED-198) 

- Se debe poder seleccionar el cliente de una lista según el repositorio clientes 


- Se debe cambiar el cliente de la orden


- Se deben actualizar todos los precios para la lista de precios del nuevo cliente


- So debo admitir la opción, si la orden esta pendiente
