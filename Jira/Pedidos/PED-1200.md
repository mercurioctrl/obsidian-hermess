## Descripción

Haciendo algunas pruebas para el MVP note que al realizar algún cambio en alguna orden, la misma recarga para poder mostrar los datos de esa orden especifica en el listado todo el repositorio de órdenes.

Se suele consultar todo el repositorio de ordenes como se inicializa por defecto, que devuelve al menos 15 órdenes con todos sus datos.

```
GET {API_URL}/v1/orders?between=11-12-2025_26-12-2025&companyCode=4&currentPage=1
```

Se debe intentar cargar la menor información posible o aplicar la mayor cantidad de filtros posible de modo tal que el costo por obtener la información requerida sea el mínimo.
