## Descripción

Crearemos una orden especifica de uso de postventa que les permita cursar mercadería nuevamente a los clientes

```
POST /v1/orders
```

```
{
  "clientId":26806,
  "branch":"0003"
}
```

Quienes poseen el rol `user.roleDescription` 'Departamento RMA' o 'Jefe de servicio post venta' solo pueden hacer ordenes de este tipo.

Y agregaremos al repositorio un nuevo tipo de pedido  'Postventa’ segun el repo

```
SELECT cobserv, orderTypeId
        FROM [NewBytes_DBF].[dbo].[pedclit]
        where orderTypeId is not null
        GROUP BY cobserv,orderTypeId
        order by orderTypeId ASC
```
