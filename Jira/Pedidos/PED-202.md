## Descripción

Esta feature se trata sobre el listado de solicitudes que hacen los clientes para darse de alta.

Cuando se crea una cuenta en la web para darle de alta, no es inmediata la creación del cliente. Esta “solicitud” para a formar parte del listado de “solicitudes” hasta que un vendedor la ve y da de alta la cuenta de la misma (Esto lo haremos en la próxima historia).



```
GET {API_URL}/v1/clientRequest
```

### Repositorio

```
SELECT
id,
UserName,
UserEmail,
correoConfirmado,
REPLACE(nombre, 'ñ', 'ñ')  as nombre,
CONVERT(VARCHAR, fecha, 20) as fecha,
RSocial,
NFantasia,
dni,
cuit,
telFi,
telCel,
email2,
wwwEmpresa,
comoNos
FROM [NB_WEB].dbo.usuarios_nb
RIGHT JOIN [NB_WEB].dbo.info_usuarios
ON usuarios_nb.UserId = info_usuarios.UserLogId
WHERE activa = 0 and eliminado =0
ORDER BY UserId DESC
```
