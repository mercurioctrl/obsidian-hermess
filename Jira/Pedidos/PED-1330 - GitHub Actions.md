## Descripcion

Definir e implementar validaciones automáticas dentro de GitHub Actions para fortalecer el flujo de despliegue y revisión de cambios.

Esta historia contempla la incorporación progresiva de pruebas y validaciones en el pipeline, con el objetivo de detectar errores oportunamente, dar mayor visibilidad al resultado de los despliegues y sentar la base para futuras validaciones más robustas.
