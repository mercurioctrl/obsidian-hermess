## Descripción

**Qué hay que hacer**

- Crear una asignación (presupuesto máximo) de un fondo a una acción.


- Actualizar `Marketing_FundStats.AllocatedTotal` y `Marketing_ActionStats.AllocatedTotal` en la misma transacción.



**Endpoint**

```
POST /v1/marketing/allocations
```

**Ejemplo request (payload)**

```
{
  "fundId": 101,
  "actionId": 501,
  "amountAllocated": 1200.00
}

```

**Ejemplo response (201)**

```
{
  "id": 9001,
  "fundId": 101,
  "actionId": 501,
  "amountAllocated": 1200.00,
  "createdAt": "2026-01-05T06:18:00-03:00"
}

```

**SQL Server**

- `NB_WEB.dbo.Marketing_Allocations`


- `NB_WEB.dbo.Marketing_FundStats`


- `NB_WEB.dbo.Marketing_ActionStats` (PK `ActionId + Currency`)



**Criterios de aceptación**

- Valida existencia de `FundId` y `ActionId`.


- Valida `AmountAllocated > 0`.


- No permite asignar más que el disponible del fondo: `AmountOriginal - AllocatedTotal >= AmountAllocated`.


- Actualiza `FundStats` y `ActionStats` en la misma transacción.


- Si existe `UQ (FundId, ActionId)`, un duplicado devuelve `409` (o `400`) con mensaje claro.





**SQL Server (tablas)**

### `NB_WEB.dbo.Marketing_Allocations`

- `Id` int IDENTITY PK


- `FundId` int NOT NULL FK → `Marketing_Funds(Id)`


- `ActionId` int NOT NULL FK → `Marketing_Actions(Id)`


- `AmountAllocated` decimal(18,2) NOT NULL


- `CreatedAt` datetime2 NOT NULL DEFAULT sysdatetime()


- `CreatedByUserId` int NULL



Restricciones/índices:

- `CK_MAlloc_Positive` CHECK (`AmountAllocated > 0`)


- `UQ_MAlloc_Fund_Action` UNIQUE (`FundId`,`ActionId`) *(1 asignación por par; si querés “incrementos”, se cambia a PATCH o se permite múltiple)*


- `IX_MAlloc_FundId` (`FundId`)


- `IX_MAlloc_ActionId` (`ActionId`)



### `NB_WEB.dbo.Marketing_ActionStats` *(resumen por acción y moneda del fondo)*

- `ActionId` int NOT NULL FK → `Marketing_Actions(Id)`


- `Currency` char(3) NOT NULL *(ARS/USD)*


- `AllocatedTotal` decimal(18,2) NOT NULL DEFAULT 0


- `SpentTotal` decimal(18,2) NOT NULL DEFAULT 0


- `UpdatedAt` datetime2 NOT NULL DEFAULT sysdatetime()


- PK compuesta: (`ActionId`,`Currency`)
