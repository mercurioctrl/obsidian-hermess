## Descripción

### Cancela una reserva existente sin borrarla físicamente.
Cambia su estado a `cancelled` y la excluye automáticamente de las vistas del calendario.

```
DELETE /v1/reservations/{reservationId}
```

**Response 200**

```
{ "data": { "id": 124, "status": "cancelled" } }

```

**Reglas**

- Setear `status='cancelled'` y `cancelled_at=now()`.


- (Opcional) Autorizar: solo creador o admin.



**Sql Orientativo**

```
UPDATE dbo.room_reservations
SET status='cancelled',
    cancelled_at = SYSDATETIMEOFFSET(),
    updated_at = SYSDATETIMEOFFSET()
WHERE id = @reservationId AND status='active';
```
