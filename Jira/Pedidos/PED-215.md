## Descripción

Lo que se busca con esta feature es generar un listado de productos y su informacion para obtener informacion relevante de como fueron sus métricas de venta y tomar decisiones comerciales.

```
GET {API_URL}/v1/reports/stocks
```

- **FECHA DE INGRESO**: 


- **MARCA**:  


- **Categoría**: 


- **Sub Categoría**: Este por ahora estará en blanco


- **ID NB**:  


- **DESCRIPCION**:  


- **STOCK 0**: Cantidad inicial de stock separado en un deposito invisible


- **STOCK**: Cantidad actual de stock disponible.


- **STOCK LO**: Stock de libre opcion


- **TOTAL**: Cantidad total de stock, sumando todas las columnas que refieren a el


- **DISPONIBLE**: Cantidad de stock disponible una vez que se restan los pedidos que ya estan confirmados


- **ORDEN**: Productos que han sido ordenados pero aún no han llegado o no han sido procesados.


- **PEDIDO**: Pedidos realizados por clientes que aún no han sido atendidos.


- **RESERVA**: Productos apartados para pedidos específicos, no disponibles para otras ventas.


- **FACTURADOS**: Productos vendidos (ojo con esto, no refiere a algo fiscal, sino a ventas)


- **FOB**: Precio de compra sin impuestos


- **COSTO**: Precio de adquisición o costo de producción del producto.


- **MAY**: Precio de venta al por mayor.


- **LO**: Precio de libre opcion



### Repositorio orientador

```
SELECT FORMAT(ULTIMO_INGRESO, 'dd-MM-yy') AS FECHA_INGRESO
    , marcas.referencia AS MARCA
    , familias.DESCRIPCION AS CATEGORIA
    , articulo.ID_ARTICULO AS ID_NB
    , articulo.cDetalle AS DESCRIPCION
    , stocks.nstock_d1 AS STOCK_0
    , stocks.nstock AS STOCK
    , stocks.nstock_lo AS STOCK_LO
    , (nstock + nstock_lo + nstock_en_cola + nstock_d1) AS TOTAL_STOCK
    , (
        (nstock + nstock_lo + nstock_en_cola + nstock_d1) - (
            SELECT SUM(ncanped)
            FROM [NewBytes_DBF].[dbo].pedclil pl_sub
            LEFT JOIN NewBytes_DBF.dbo.pedclit pt_sub
                ON pl_sub.cnumsuc = pt_sub.cnumsuc
                    AND pl_sub.cnumped = pt_sub.cnumped
            WHERE (
                    pt_sub.cobserv = 'INTERNO'
                    OR pt_sub.cobserv = 'DESCARGADO'
                    )
                AND pt_sub.cestado = 'P'
                AND (
                    pl_sub.cref = articulo.cRef
                    OR pl_sub.ID_Articulo = articulo.ID_ARTICULO
                    )
            )
        ) AS DISPONIBLE
    , (
        (
            SELECT SUM(ncanped)
            FROM [NewBytes_DBF].[dbo].pedclil pl_sub
            LEFT JOIN NewBytes_DBF.dbo.pedclit pt_sub
                ON pl_sub.cnumsuc = pt_sub.cnumsuc
                    AND pl_sub.cnumped = pt_sub.cnumped
            WHERE pt_sub.cestado = 'P'
                AND (
                    pl_sub.cref = articulo.cRef
                    OR pl_sub.ID_Articulo = articulo.ID_ARTICULO
                    )
            )
        ) AS PEDIDO
    , (
        (
            SELECT SUM(ncanped)
            FROM [NewBytes_DBF].[dbo].pedclil pl_sub
            LEFT JOIN NewBytes_DBF.dbo.pedclit pt_sub
                ON pl_sub.cnumsuc = pt_sub.cnumsuc
                    AND pl_sub.cnumped = pt_sub.cnumped
            WHERE (
                    pt_sub.cobserv = 'INTERNO'
                    OR pt_sub.cobserv = 'DESCARGADO'
                    )
                AND pt_sub.cestado = 's'
                AND (
                    pl_sub.cref = articulo.cRef
                    OR pl_sub.ID_Articulo = articulo.ID_ARTICULO
                    )
            )
        ) AS RESERVA
    , (
        SELECT sum(albclil.ncanent) AS RESERVAS_FACTURADAS
        FROM NewBytes_DBF.dbo.albclil
        LEFT OUTER JOIN NewBytes_DBF.dbo.albclit
            ON NewBytes_DBF.dbo.albclil.ID_NROREMCLI_ENC = NewBytes_DBF.dbo.albclit.ID_NROREMCLI_ENC
        WHERE (
                albclil.cref = articulo.cRef
                OR albclil.ID_Articulo = articulo.ID_ARTICULO
                )
            AND (
                NewBytes_DBF.dbo.albclit.lfacturado = 1
                OR albclit.ntipoalb > 1
                )
        ) AS FACTURADOS
    , isnull((
            SELECT sum(albclil.ncanent)
            FROM NewBytes_DBF.dbo.albclil
            LEFT JOIN NewBytes_DBF.dbo.albclit
                ON albclit.ID_NROREMCLI_ENC = albclil.ID_NROREMCLI_ENC
            WHERE albclil.cref = articulo.cref
                AND dfecalb >= Dateadd(Month, Datediff(Month, 0, DATEADD(m, - 1, current_timestamp)), 0)
            ), 0) AS ventas30DIAS
    , isnull((
            SELECT sum(albclil.ncanent)
            FROM NewBytes_DBF.dbo.albclil
            LEFT JOIN NewBytes_DBF.dbo.albclit
                ON albclit.ID_NROREMCLI_ENC = albclil.ID_NROREMCLI_ENC
            WHERE albclil.cref = articulo.cref
                AND dfecalb >= Dateadd(Month, Datediff(Month, 0, DATEADD(m, - 2, current_timestamp)), 0)
            ), 0) AS ventas60Dias
    , articulo.ID_PRODUCTO AS SKU
    , ncosteprom AS COSTO
    , (
        SELECT TOP (1) nprediv AS COSTO_FOB
        FROM newbytes_dbf.dbo.albprol
        LEFT JOIN NewBytes_DBF.dbo.albprot
            ON NewBytes_DBF.dbo.albprol.nnumalb = NewBytes_DBF.dbo.albprot.nnumalb
        WHERE (
                albprol.cref = articulo.cRef
                OR albprol.ID_Articulo = articulo.ID_ARTICULO
                )
        ORDER BY dfecalb DESC
        ) AS FOB
    , articulo.npvp5 AS MAY
    , articulo.nplo AS LO
FROM NewBytes_DBF.dbo.articulo
LEFT JOIN NewBytes_DBF.dbo.stocks
    ON NewBytes_DBF.dbo.articulo.ID_ARTICULO = NewBytes_DBF.dbo.stocks.ID_ARTICULO
LEFT JOIN NB_WEB.dbo.marcas
    ON marcas.id = articulo.Id_Marca
LEFT JOIN NewBytes_DBF.dbo.familias
    ON familias.ID_FAMILIA = articulo.ID_FAMILIA
WHERE articulo.EXCLUIR <> 1

```
