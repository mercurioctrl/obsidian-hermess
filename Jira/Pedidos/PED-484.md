[attachment]
 

Solo deben mostrarse aquellas solicitudes donde el cliente aun no fue creado. 



Para poder hacer esto es necesario marcarla de la siguiente forma, al crear un cliente , similar a como hicimos en [link](https://lioteam.atlassian.net/browse/PED-205) 

```
UPDATE [NB_WEB].[dbo].[info_usuarios] SET eliminado = 1 where id = ?
```

Solo que esta vez, se hace al crear un cliente, que viene de una solocitud

[attachment]
Si no es posible enlazar la tabla `[NB_WEB].[dbo].[info_usuarios]` debemos enviar el id de la misma de manera “oculta” al levantar el modal de crear cliente cuando viene de una solicitud. Si tienen dudas con esto me dicen.
