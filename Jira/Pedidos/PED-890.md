## Descripción

```
POST {{API_URL}}/v1/syncUp/unlinkInactiveClients
```

```
{
  token: {token}
}
```

**Query para respaldar y liberar clientes:**

```
UPDATE [NewBytes_DBF].[dbo].[clientes]
SET 
    ULTIMO_VENDEDOR = ID_VENDEDOR,   -- Respaldo del vendedor actual
    FECHA_CAMBIO_VENDEDOR = GETDATE(), -- Fecha del cambio
    ID_VENDEDOR = NULL, -- Liberación del cliente
    ccodage = NULL -- La clave vieja que identifica al vendedor y aun se usa
WHERE ULTIMA_COMPRA < DATEADD(DAY, -180, GETDATE());
```
