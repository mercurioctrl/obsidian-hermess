## Descripción

Con el objetivo de permitir la modificación exclusiva de los parámetros del cliente, se desarrolló un nuevo recurso al cual deberá dirigirse la actualización.

En lugar de utilizar:

```
GET {API_URL}/v1/clients/{clientId}
```

Se deberá apuntar a:

```
PATCH {API_URL}/v1/clients/{clientId}/params
{
    "currencyId": 1,
    "companyCode": 11,
    "salespersonId": 72,
    "specialPrice": 0,
    "paymentTerms": 15
}
```
