## Descripción

Agregaremos los recursos para gestionar las intervenciones técnicas necesarias antes de enviar un pedido. Esto incluye la configuración de indicadores para acciones como “armado de PC”, “actualizar la BIOS” e “instalar el sistema operativo”. El recurso operará sobre la tabla `[NewBytes_DBF].[dbo].pedclit` a la que agregaremos 3 flags nuevas

- assemblePc


- updateBios


- installOS



```
GET {API_URL}/v1/orders/{branch}-{order}/technicalIntervention
```

```
{
  "orderNumber": "10405377",
  "branchNumber": "0002",
  "assemblePc": true,
  "updateBios": false,
  "installOS": true
}

```

```
PATCH /v1/orders/{branch}-{order}/technicalIntervention
```

```
{
  "assemblePc": true,
  "updateBios": false,
  "installOS": true
}
```

### Criterios de aceptación 

- **GET** devuelve correctamente los flags técnicos (`assemblePc`, `updateBios`, `installOS`) para un `order` y `branch` válido.


- **PATCH** actualiza solo los campos enviados y respeta los no enviados.


- Se devuelve **404** si el `orderId` no existe.


- Se devuelve **400** si el body está vacío o malformado.


- Los valores se reflejan correctamente en la tabla `[pedclit]`.
