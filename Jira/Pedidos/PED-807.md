## Descripción

Dado que las motos o los transportes a veces cobrar una espera o estacionamiento para algunas zonas particulares, surgio la idea de agregar para esos casos la posibilidad de recibir un costo adicional

De esta forma si nos llega el parámetro `shippingCost` distinto de `0` o distinto de `null` entonces simplemente lo sumaremos antes de guardarlo en pedclil.npreunit y pedclil.ncostoextra

`shippingCost` =`shippingCost`+ `additionalCost`

`shippingPrice` =`shippingPrice`+ `additionalCost`

```
POST {API_URL}/v1/orders/{order}/addShipping
```

```
{
  "shippingMethodId":3030,
  "shippingName":"Moto (Capital Federal)",
  "shippingPrice":5700,
  "additionalCost": 500.55,  < SE AGREGA NUEVO PARAMETRO
  "shippingCost":5700,
  "customerAddressId":18063
}
```

Por el momento solo aplica para moto o camioneta
