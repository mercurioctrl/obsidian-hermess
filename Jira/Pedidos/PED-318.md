## Descripción

Cuando nos mandan una orden a través de algún portal y esta por alguna razón no pude satisfacer con el stock la demanda del cliente, entonces  recibimos lo siguiente

```
{API_URL}/v1/downloadOrder/{pedido}
```

```
{
    "errors": {
        "status": 401,
        "title": "No hay stock suficiente para el articulo 103506, quedan 573",
        "file": "\/var\/www\/app\/app\/Services\/Order\/OrderService.php",
        "line": 169
    }
}
```

Cuando esto sucede, es correcto mostrar el mensaje de error. 
Pero al mismo tiempo debemos abrir la orden para poder editarla.
