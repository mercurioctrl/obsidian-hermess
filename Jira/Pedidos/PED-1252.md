## Descripción

Realizaremos un refactor para que el valor de la clave `currencyId` sea `Marketing_Funds.CurrencyId` y agregaremos la clave `currencyDescription` de `NewBytes_DBF.dbo.FP_Monedas.Descripcion`.

```
GET /v1/marketing/funds
```

            "currencyDescription": "ARS" <------------------ SE AGREGA CLAVE Y VALOR

Adicionalmente, se incluirá un nuevo filtro llamado `search`, que servirá para realizar búsquedas por texto libre en `Marketing_Funds.Name`, y el filtro `between`, para buscar entre fechas de creación en `Marketing_Funds.CreatedAt`.

```
GET /v1/marketing/funds?between=29-12-2025_13-01-2026&search=prueba
```
