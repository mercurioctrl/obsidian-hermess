## Descripción

Tanto en el response del POST como en el GET se debe agregar  `notes`

```
POST /v1/rooms/{roomId}/reservations
```

**Response **

```
{
  "data": {
    "id": 124,
    "roomId": 1,
    "title": "Reunión producto",
    "startAt": "2025-12-16T10:30:00-03:00",
    "endAt": "2025-12-16T11:15:00-03:00",
    "status": "active",
    "createdByUserId": 55,
    "notes":"aqui las notitas",// --> Se debe agregar
  }
}

```

```
GET /v1/rooms/{roomId}/reservations?from={iso}&to={iso}
```

**Response **

```
{
  "data": [
    {
      "id": 123,
      "roomId": 1,
      "title": "Sync",
      "startAt": "2025-12-16T10:00:00-03:00",
      "endAt": "2025-12-16T11:00:00-03:00",
      "status": "active",
      "createdByUserId": 55,
      "createdAt": "2025-12-14T20:10:00-03:00",
       "notes":"aqui las notitas",// --> Se debe agregar
    }
  ]
}

```
