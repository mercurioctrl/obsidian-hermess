## Descripción

Existe una oportunidad de mejora para ser mas específicos con el estado de un pedido. Esto surge a partir de un caso puntual que venia siguiendo para saber como era cada paso. 

Se trata del pedido X000200583526 / 0002-10355300 / LO - 650259 

Este pedido llego a deposito y fue acreditado por el mismo (ya sea de manera completa, o parcialmente pero seleccionando todos los items, osea completo)

El crédito se hizo PERFECTO

La mercedaria se acredito PERFECTA

El único problema que veo es con el estado final en el que termino el pedido: ARMADO FINALIZADO


Es raro que quede en este estado, porque al no serializarse nunca no se pudo haber armado. Y por otro lado el estado final para este caso debería ser: 15 - Acreditacion completa
