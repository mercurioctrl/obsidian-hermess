## Descripción

```
GET {API_URL}/v1/shipments/X000200547628/sellerComments
```

Basándonos en un rauta como esa, agregaremos un recurso para obtener los comentarios del pedido dejados por el vendedor.

Se mostraran dos comentarios de la siguiente forma

```
{
commentOrder: {comentario que surge de la primera query}
commentSellerClosing: {comentario que surge de la segunda query}
}
```

el primero basado en 

```
[NB_WEB].[dbo].[comentariosPedidos] // esta basado en pedclit.cnumped
```

y el segundo en

```
[NEW_BYTES].[dbo].[MS_REMITO_CABECERA] esta basado en albclit.CNUMALB y ablclit.cnumsuc
```

si existiera..
