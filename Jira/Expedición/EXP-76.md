## Descripción

Dentro del Micro servicio de envíos, existen recursos específicos para poder descargar etiquetas en los formatos clásicos.

Se debe buscar los patrones comunes entre estas para generar una etiqueta que sirva para todos los casos de ser posible.

En caso contrario, pueden hacerse plantillas separadas.

Esta historia trata sobre el research necesario para entender que datos son necesarios para cada etiqueta y asegurarse que todos los datos esten para todos los casos.
