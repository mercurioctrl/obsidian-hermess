## Descripcion

Es necesario verificar en el sistema de gestión tanto el recurso que proporciona los datos como el que procesa los créditos. Esto es para asegurar que, cuando se realiza un crédito o reembolso de algo que inicialmente se facturó con percepciones incluidas, estas percepciones sean correctamente reflejadas en el crédito fiscal y registradas en la cuenta corriente.
