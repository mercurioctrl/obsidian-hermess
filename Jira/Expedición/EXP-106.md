## Descripción

Solo cuando el pedido esta sin despachar se pueden remover los números de serie por si se equivocan, del pedido.

Para esto se debe hacer lo inverso a [link](https://lioteam.atlassian.net/browse/EXP-66) para removerlo.

```
DELETE {API_URL}/v1/orders/{pedido}/serials/{itemId}
```

**Carga útil**

```
[
  {
    "serials": [
        'FAT43939393933',
        'FAT43939393933',
        'FAT43939393933',
        'FAT43939393933',
        'FAT43939393933',
        'FAT43939393933',
        'FAT43939393933',
    ]
}
]
```
