## Descripción

Agregaremos `stockWarehouseId` como un parámetro opcional, el mismo se obtiene de` /v1/providersOrders/{orderId}` que en ese punto ya lo tenemos en pantalla.

En caso de no estar disponiAgregaremos `stockWarehouseId` como un parámetro opcional para guardar en `[NEW_BYTES].[dbo].[ST_DETALLE_STOCK].ID_ALMACEN`

- Si el parámetro `stockWarehouseId` viene en `NULL` o bien el front no lo envía, entonces se guarda `ID_ALMACEN=2`


- Si no esta presente el parámetro, el recurso no rompe, usa el valor por defecto



```
POST /v1/providersOrders/{id}/serials/{itemId}
```

```
{
"mode":"list",
"stockWarehouseId": 2, <-- Se agrega como parametro opcional

]
}
```
