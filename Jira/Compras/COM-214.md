## Descripción

revisar el parametro `countryId`
Ejemplo payload de de creacion


```
{
    "name": "msi co",
    "code": "mso",
    "address": "",
    "countryId": 13,// se envia 
    "provinceId": null,
    "cityId": null,
    "default": false,
    "phone": ""
}
```

y al crearse devuelve


```
{
    "id": 9,
    "code": "mso",
    "name": "msi co",
    "address": "",
    "phone": "",
    "default": false,
    "cityCode": "",
    "cityId": 0,
    "cityName": "",
    "provinceId": 0,
    "provinceCode": 0,
    "provinceName": "",
    "countryDescription": "",
    "countryId": 0, //  Viene siempre en 0
    "prefixFlag": "",
    "companyCode": 0
}
```
