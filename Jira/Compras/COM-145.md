## Descripción

Agregaremos un recurso que nos permita editar los impuestos distribuidos que tiene una orden determinada.

Para eso enviaremos solo el objeto `distributeTaxes` con la informacion necesaria de la siguiente forma, alterando o creando los registros dentro de `[NewBytes_DBF].[dbo].[pedproi]`

```
PATCH {API_URL}/v1/providerOrder/11368
```

```
"distributeTaxes": [  <<<<<-----------------NUEVO
      {
      "acronym": "DDI",
      "description": "Derechos de importancion",
      "rate": 100.0,
      "taxBase": 895.18,
      "id": 2
      },
    {
      "acronym": "IB",
      "description": "Percepcion ingresos brutos",
      "rate": 6.0,
      "taxBase": 12525.34,
      "id": 1
      },              
  ],
```

[attachment]
Se desacopla la logica de agregar o editar los impuesto distribuidos a un nuevo recurso `PATCH {API_URL}/v1/providerOrder/11368/distributeTaxes`
