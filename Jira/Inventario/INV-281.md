## Descripción

Al agregar un kit, este debe tener un almacén o deposito especifico, para esto debemos agregar el parámetro `stockWarehouseId`a la tabla `[NewBytes_DBF].[dbo].[articulo_kits]`

Este parametro lo recibiremos de parte del front en el recurso 

```
PATCH {API_URL}/itemsKits/{itemId}
```

```
{
  "itemIdInKit":106745,
  "quantityNeeded":2,
  "stockWarehouseId":2 <--  Se agrega
}
```
