## Descripción

Actualmente, la integración con el recurso de sugerencia de descripción está generando errores al realizar solicitudes al siguiente endpoint:

```
POST {API_URL}/suggestDescription
```

Este recurso parece estar fallando o respondiendo incorrectamente, lo cual interrumpe el funcionamiento esperado. Mientras se investiga y se encuentra una solución definitiva, es necesario **desconectarlo temporalmente** para evitar errores en el sistema y mantener una experiencia estable para el usuario.
