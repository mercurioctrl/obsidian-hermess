## Descripcion

Crearemos una nueva pestaña para las “Descripciones Autogeneradas”

```
GET {API_URL}/v1/autoGeneratedDescription?description={nombre de producto}&karma=1&categoryId={categoryId}&brandId={brandId}&stock=1
```

Estará filtrado por defecto en aquellas que queremos revisar, es decir donde `karma=0`
Agreagremos los filtros para 

- karma


- categoria


- marca


- Nombre





Ademas debe poder editarse con el recurso [link](https://lioteam.atlassian.net/browse/INV-82)
