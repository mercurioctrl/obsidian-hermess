## Descripcion

Crearemos un recurso para mostrar las descripciones según algunos filtros para posteriormente revisar las que tienen karma negativo y poder editarlas o pedir que se vuelva a generar.

```
GET {API_URL}/v1/autoGeneratedDescription?description={nombre de producto}&karma=1&categoryId={categoryId}&brandId={brandId}&stock=1
```

- description: permite buscar por el nombre del producto


- karma: cuando es 1, muestra los que tienen karma mayor o igual a cero. Cuando es 0, muestra los que tienen karma negativo. . Si no viene, muestra ambos


- CategoriId: filtro de cateogria


- brandId: Filtro de marca


- stock: Cuando es 1, muestra productos con stock, cuando es cero, sin stock. Si no viene, muestra ambos
