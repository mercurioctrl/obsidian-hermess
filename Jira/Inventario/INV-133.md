Debemos tener en cuenta que el IVA no se guarda ni puede editar de forma lineal, sino que se guarda en los parametros  `ctipoiva y ctipoivac` respectivamente y con una letra.

Cuando el iva es 10.5, usamos 'M'

Cuando el iva es 21, usamos 'G'

Ejemplos de tipos de IVA

```
SELECT TOP(1) ivaCompra,ivaVenta,ctipoiva,ctipoivac
  FROM [NewBytes_DBF].[dbo].[articulo]
  WHERE ivaCompra = 10.5;
  
  SELECT TOP(1) ivaCompra,ivaVenta,ctipoiva,ctipoivac
  FROM [NewBytes_DBF].[dbo].[articulo]
  WHERE ivaCompra = 21;
```
