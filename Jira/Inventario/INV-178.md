## Descripción

Lo que se busca es poder cambiar la clave (el nombre) de un atributo, asi como tambien puedo cambiar el valor.

Esto me permite arreglar un atributo mas rápido que teniendo que volverlo a cargar y es muy útil para perfeccionar el etiquetado de atributos.

(Ojo, tal vez eso el recurso ya lo hace, en ese caso marcar como realizada)

```
PATCH {API_URL}/item/{itemId}
```

```
{
"attributes":
  [
    {
    "name":"Tipo de memoria RAM", <-- Este es el que cambia
    "value":"DDR6",
    "id":793371,
    "required":0,
    "videoUrl":""
    }
 ]
}
```
