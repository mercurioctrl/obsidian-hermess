## Descripción

Hoy en día se utiliza `[NB_WEB].[dbo].[familias].mostrar` para mostrar o no el ítem , pero el criterio es que esto es para el sistema como se hace en marcas.

Para esto lo primero que haremos es agreagar un filtro para ver los que esta o no ocultos segun este parametro.

```
GET {API_URL}/categories?hide=true/false/null
```

Cuando `hide=true` entonces `[NB_WEB].[dbo].[familias].mostrar`=0

Cuando `hide=false` entonces `[NB_WEB].[dbo].[familias].mostrar`=1

Cuando `hide` esta ausente entonces muestro todos
