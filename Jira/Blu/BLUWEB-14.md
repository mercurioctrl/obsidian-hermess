## Descripción

Como se implementó en [link](https://www.nb.com.ar/vcard/nsheridaim@nb.com.ar) , se debe crear un concepto visual de cómo se verá una vCard dentro del sitio al ingresar un correo electrónico.
**No es necesario diseñar cada caso particular**, solo se solicita **una propuesta general** para definir estilo visual (colores, composición, tamaños, tipografía, etc.).

El objetivo es establecer una referencia base para luego replicar la lógica con otros correos.

**Criterios de aceptación:**

- Presentar una propuesta visual o herramienta similar.


- Incluir al menos un ejemplo de vCard (puede ser con datos ficticios).


- Definir uso de colores, proporciones y estilo general.


- Enfocarse en transmitir identidad de marca y claridad visual.
