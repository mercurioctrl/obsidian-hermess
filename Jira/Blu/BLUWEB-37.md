## Descripción

Vamos a iniciar el desarrollo del sitio web frontend utilizando el repositorio [blu-web-v1](https://github.com/BluIncStudio/blu-web-v1), tomando como referencia el diseño en Figma:
🔗 [Figma - Diseño BLU](https://www.figma.com/design/ObV2dZ2eg78sy3cKpJtPff/BLU?node-id=0-1&p=f&t=Q5jHKZeuTv2yvbRN-0)

Se coordina una reunión para ajustar detalles visuales según el diseño en Figma.

Para esta etapa, trabajaremos sobre un esqueleto base compuesto por las siguientes secciones:

- Header


- Cuerpo principal


- Sección con divisiones de la compañía


- Mensaje con Call to Action para contacto


- Footer

**Requerimientos funcionales:**

- Implementar el modo claro/oscuro, donde el negro y el blanco se invierten según el modo. Los demás colores deben mantenerse iguales.


- Al hacer scroll, debe haber transiciones suaves entre secciones, alternando entre fondo negro y blanco según corresponda.


- Usar como base el repositorio indicado y mantener el código modular para facilitar los cambios en el diseño final.



**Notas adicionales:**
Esto puede ser difícil de explicar sin verlo, por lo que necesitamos una reunión para alinear detalles. Avisame cuándo podés así lo vemos juntos.

**Criterios de aceptación:**

- Se genera la estructura inicial del sitio con las secciones mencionadas.


- El cambio entre modo claro/oscuro funciona correctamente.


- Las transiciones al hacer scroll cambian de fondo negro a blanco según el diseño.


- El código se sube al repositorio `blu-web-v1` en una rama propia (`feature/estructura-inicial`).
