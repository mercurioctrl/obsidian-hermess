## Descripción

En la pestaña de recuperos debemos:

- Mostrar el filtro de fecha predeterminado ( ultimos 6 meses) ‘between’


- Mostrar el filtro de proveedor predeterminado ( Deposito 2 - servicio tecnico ) `warehouseId` = 2


- Falta agregar el filtro de Enviado a proveedor (por default en no ) `sendToProvider`= 0





Tambien se debe revisar que cuando se agregan filtros en la pestaña de recuperos se rompe el front al cambiarse a envios a proveedor y el search de proveedor no funciona (avisame si es de back)
