## Descripción

Maer en base de datos interna (`newbytes_Dbf.dbo.pedclit.tiSucces`) la fecha en la que se completa la intervención técnica, utilizando los campos `order` y `branch` como referencia.

```
PATCH {API_URL}/v1/tiSucces
```

```
{
  "order": 10408607,
  "branch": 0002,
}
```

```
{
  "message": "Intervención técnica marcada como completa",
  "order": 10408607,
  "branch": 0002,
  "fechaMarcada": "2025-05-22T14:00:00"
}
```
