## Descripcion

Si el técnico esta sin asignarse, cualquier usuario se puede auto asignar.

Una vez asignado el técnicos, solo quien tiene permisos, puede asignarselo a alguien.

Ademas, este ultimo rol, puede asignar a cualquier tecnico.
