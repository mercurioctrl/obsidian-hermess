## ¿Qué es un Sandbox y Cómo Utilizarlo con los Recursos Gamma?

Un sandbox, en el contexto de desarrollo de software, es un entorno aislado y seguro donde los desarrolladores pueden probar y experimentar con aplicaciones, servicios o sistemas sin afectar el entorno de producción. Es decir que pueden realizar todas las operaciones en un entorno artificial. Pueden probar nuevas ideas, integraciones y funcionalidades sin riesgo de generar pedidos, reservas u operaciones reales, lo que significa que los cambios realizados en el sandbox no afectarán a los sistemas en vivo.

Utilizando los Recursos Gamma en el Sandbox:

En tu sandbox, cuentas con dos recursos principales:

### [https://gamma.api.nb.com.ar](https://gamma.api.nb.com.ar) (Back-end):

Aquí se encuentran alojadas las API, que cuenta con todos los mismos recursos que cuenta la api de producción. Podes utilizar este recurso para probar y desarrollar nuevas funcionalidades, realizar integraciones con otros sistemas, o para simular el comportamiento del back-end en diferentes escenarios.

### [link](https://gamma.nb.com.ar)  (Front-end): Este recurso corresponde al front-end de tu aplicación o sistema. Este es el sandbox del sitio web, que podes utilizar para ver como impacta en tu cuenta las acciones que realizas en el sandobox API.

Escenarios de Uso:

Desarrollo de Nuevas Funcionalidades: Utiliza el sandbox para desarrollar y probar nuevas funcionalidades antes de implementarlas en el entorno de producción.

Pruebas de Integración: Utiliza el sandbox para realizar pruebas de integració, así como con otros servicios y sistemas externos.

Experimentación y Prototipado: Utiliza el sandbox para experimentar con nuevas ideas, prototipar nuevas características, y explorar diferentes enfoques sin afectar el entorno de producción.

En resumen, un sandbox es una herramienta invaluable para el desarrollo de software que proporciona un entorno seguro y controlado para probar, experimentar y desarrollar aplicaciones y sistemas de manera eficiente y sin riesgos.
