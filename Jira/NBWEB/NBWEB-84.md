## Descripción

Dentro del .env y .env-example debe existir el parametro DEBUG que se marca en uno para ver los reportes de error dentro de un dto de retorno o incluso para activar el reporte de errores de php.

El fin de esta terea, es activar para toda la api este parametro.

En producción la misma debe ser cero, y no mostrar ningun error especifico.
