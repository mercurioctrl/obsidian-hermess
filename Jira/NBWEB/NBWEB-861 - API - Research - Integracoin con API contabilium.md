## Descripcion

Se debe investigar sobre la posibilidad de importar el catalogo a [Contabilium](https://contabilium.com/ar/)

Al parece este sistema es el ERP online mas usado de Argentina y tiene una API que permite empujarle los productos (y actualizar stock y precio).

Se debe verificar que requisitos son necesarios a nivel de credenciales y que recursos es posible utilizar para 

1 - Cargar el catalogo

2 - Actualizar precios y stock de manera periódica

3- Evaluar si es posible en u futuro (menos relevante) traer las compras para realizar la orden automatica.
