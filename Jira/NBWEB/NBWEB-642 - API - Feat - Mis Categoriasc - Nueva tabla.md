## Descripcion

Crearemos la tabla `[NB_WEB].[dbo].[userCategories]` con las siguientes columnas

- id (auto)


- userId (es el id del usuario de la api)


- categoryId (es la categoria original)


- utility (0, es un porcentaje de utilidad)


- description (Null, es para poner un nombre alternativo a la categoria)
