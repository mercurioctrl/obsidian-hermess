## Descripcion

Agregaremos una pagina principal al sitio developers (home) para explicar para que sirve el sitio en general y a modo de presentación.

Moveremos la introducción a la Api a segunda posición



El sitio de New Bytes ofrece una plataforma exclusiva para clientes dirigida a desarrolladores, diseñada para guiarlos a través del proceso de integración y utilización de su API para ejecutar flujos de compra. Desde la autenticación inicial hasta la confirmación de pedidos, el sitio proporciona detalladamente cada paso necesario, incluyendo búsqueda de productos, gestión de carritos de compra y manejo de listas de precios, dropshipping, logística y postventa. 

A través de ejemplos de código y mejores prácticas, busca ser un recurso indispensable para aquellos que desean incorporar las funcionalidades de New Bytes en sus aplicaciones, asegurando una integración fluida y segura. Visita New Bytes para explorar la documentación completa y comenzar a integrar estas soluciones en tus proyectos.

Esperamos que este sitio sirva de guía para guiar en el proceso de configuración e implementacion.
