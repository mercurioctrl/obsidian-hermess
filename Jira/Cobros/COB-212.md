## Descripción

Estructura requerida que incorpora las 3 movilidades de movimientos bancarios.

```
[
  {
    "transferTypeId": 1, //Caja a Banco. 
    "amount": 15,
    "paymentMethodId": 2,
    "reference": "Test Dev comment2",
    "currencyQuote": 140.5,
    "originBankId": null,
    "destinyBankId": 16
  },
  {
    "transferTypeId": 1, //Caja a Banco. 
    "amount": 15,
    "paymentMethodId": 2,
    "reference": "Test Dev comment2",
    "currencyQuote": 140.5,
    "originBankId": null,
    "destinyBankId": 16
  }
  ]
```

 Solo cambia el `transferTypeId` segun la accion que se desea realizar.

1 = Caja a Banco

2 = Banco a Caja

3 = Banco a Banco.
