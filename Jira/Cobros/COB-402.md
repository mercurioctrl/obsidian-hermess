## Descripción

Este recurso trata sobre descargar el reporte completo de todos los moviemientos de un cliente determinado.

Recibe los codigos de cliente y muestra todos los movimiento ordenados de forma cronológica

El recurso puede llamarse

```
{API_URL}/v1/clients/xlsx
```

Esta basado en el recurso:

```
{API_URL}/v1/clients/
```

Y debe devolver las mismas columnas, solo que **en fichero de excel Saldos de cuenta - {dd-mm-yyyy}.xml**

Debe mostrar lo mismo que muestra el recurso, 

Cliente

Razón Social

CUIT

Percepciones

Deuda de cheque

Límite de crédito

Saldo

Ordenado por saldo  y para todos los clientes sin paginar en un excel.
