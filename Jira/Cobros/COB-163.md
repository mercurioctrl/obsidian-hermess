## Descripción

Basándonos en la tabla `[NEW_BYTES].[dbo].[MC_SALDOS_BANCOS]` agregaremos un parámetro extra al recurso para mostrar el ultimo saldo disponible.

Adicionalmente incluiremos en la tabla `[NEW_BYTES].[dbo].[BANCOS]` la columna `initialBalance` con valor inicial 0, que utilizaremos mas adelante como variable de ajuste.

```
  {
    "id": 34,
    "description": "BANCO PATAGONIA S.A.",
    "oldId": 9,
    "BalanceAmount": 122.06,
    "initialBalance": -500
  },
```
