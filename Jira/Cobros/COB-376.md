## Descripción

```
PATCH {API_URL}/v1/assignedCredit
```

Usando `[NEW_BYTES].[dbo].[MS_CTACTE_CLIENTES]` mostraremos el siguiente objeto

```
[ 
{
    "clientId": 19164,
    "usdCredit": 0.0,
    "checkCredit": 0.0,
    "dateModify": "2011-11-24T11:10:44",
    "userModify": "EXP3",
    "usdCreditPeriod": 0,
    "checkCreditPeriod": 0,
    "comment": ""
  }
]
```

Solo pueden ejecutar este recurso quienes tienen el permiso especifico editCredit en la tabla `[NB_WEB].[dbo].[permisos_agente]`. De no estar la columna debe agregarse.

Adicionalmente se debe enviar un correo a [gerencia@nb.com.ar](mailto:gerencia@nb.com.ar) con el texto “El usuario **usuario** edito el credito de la cuenta **(ID CLIENTE) NOMBRE CLIENTE. Saldo en u$s 32434.34. Saldo en Cheque: 324234.34”**

Adicionalmente debe guardarse 

- Ultima modificación


- Usuario que realizo la ultima modificación
