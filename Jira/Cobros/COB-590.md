## Descripción

Agregaremos un nuevo parámetro `idLo` para enlazar el numero de pedido de libre opción.

Adicionalmente agregaremos el hotlink para enlazarlo directamente usando el parámetro search y orderTypeId para filtrar perfecto la orden en la aplicación de ventas.

```
GET {API_URL}/v1/wallets/transactions/314704
```

```
{
    "transactions": [
        {
            "transactionId": 943129,
            "amount": 2160,
            "currency": "ARS",
            "conceptId": 475,
            "concept": "Pago Comisiones",
            "date": "2025-11-12 09:23:29",
            "transactionType": "income",
            "pending": false,
            "idLo": 767667,
            "hotLinkIdLo":"https://www.pedidos.saftel.com/orders?orderTypeId=5&currentPage=1&search=767667"
            
        }
    ],
    "pagination": {
        "total": 1,
        "offset": 0,
        "limit": 8,
        "order": "DESC"
    }
}
```

Los parámetros fijos como `https://www.pedidos.saftel.com/`  pueden venir directamente en el `.env` para tener bien construidos los enlaces en entornos dev/prod
