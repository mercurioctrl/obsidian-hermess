## Descripción

El RSI es como un medidor de "cansancio" para las acciones en la bolsa. Va de 0 a 100:

- Si el RSI está por encima de 70, es como si el corredor estuviera muy cansado (la acción está "sobrecomprada"). 


- Si el RSI está por debajo de 30, es como si el corredor tuviera mucha energía (la acción está "sobrevendida"). Podría acelerar (la acción podría subir).



Se piensa el RSI como un termómetro que te dice si una acción está "caliente" (sobrecomprada) o "fría" (sobrevendida). 

**¿que vamos a hacer?**

 Necesitamos integrar el indicador RSI para evaluar si nuestras acciones están "calientes" o "frías".
