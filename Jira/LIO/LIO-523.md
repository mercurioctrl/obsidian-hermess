## Descripción

Retomando lo realizado en [link](https://bluinc.atlassian.net/browse/LIO-492) 

Ahora si remplazaremos la informacion del producto “simulada” por informacion real de un producto.

Actualmente el endpoint 

```
GET /v4/yt/shorts
```

 retorna un objeto item hardcodeado (mock) en la respuesta. Se requiere refactorizar para permitir la vinculación opcional entre videos de YouTube Shorts y productos del catálogo (`CS.dbo.productos`).
La vinculación debe ser flexible, permitiendo asociar un video con:

- Un producto específico (vía item_id)


- Varios productos que comparten el mismo internal_id (id_interno)



---

Criterios de Aceptación

- Base de datos: Crear tabla LO.dbo.yt_video_products para gestionar la relación video-producto


- Backend: Modificar YoutubeShortsService para obtener datos reales del producto desde CS.dbo.productos


- Backend: Actualizar YoutubeShorts controller para eliminar el método fakeItem() y usar datos reales


- Opcionalidad: Los videos pueden existir sin productos asociados (relación no obligatoria)


- Respuesta: El endpoint debe retornar el objeto item con datos reales cuando exista vinculación, o null cuando no haya producto asociado



---

Ejemplo de una posible query utilizable

```
WITH ProductosFiltrados AS (
    -- Subquery para obtener solo el producto más nuevo por id_interno
    SELECT 
        p.id_interno,
        p.id AS productId,
        p.sku,
        p.titulo AS productTitle,
        p.precioDolar,
        p.vendedorID,
        p.envioGratis,
        p.instantFlash,
        p.activo,
        p.ocultar,
        p.fechaCreacion,
        ROW_NUMBER() OVER (
            PARTITION BY p.id_interno 
            ORDER BY p.fechaCreacion DESC, p.id DESC
        ) AS rn
    FROM CS.dbo.productos p
    WHERE p.activo = 1 
      AND p.ocultar = 0
),
ProductosConImagen AS (
    -- Agregar imagen del producto (foto portada)
    SELECT 
        pf.*,
        fotos.checksum AS productImage
    FROM ProductosFiltrados pf
    LEFT JOIN CS.dbo.productosFotos pfotos 
        ON pfotos.id_producto = pf.productId 
        AND pfotos.portada = 1
    LEFT JOIN PRODUCTOS.dbo.fotos 
        ON fotos.id = pfotos.id_foto
    WHERE pf.rn = 1
),
ProductosCompletos AS (
    -- Agregar precio y datos del vendedor
    SELECT 
        pci.*,
        (pci.precioDolar * c.cotizacion) AS finalPrice,
        (pci.precioDolar * c.cotizacion) AS listPrice,
        0 AS discount,
        v.id AS sellerId,
        v.nombre AS sellerName
    FROM ProductosConImagen pci
    INNER JOIN LO.dbo.vendedores v 
        ON pci.vendedorID = v.id 
        AND v.activo = 1
    CROSS JOIN (
        SELECT cotizacion 
        FROM NEW_BYTES.dbo.MS_COTIZACIONES 
        WHERE id = 3
    ) c
)
SELECT 
    v.id AS videoId,
    v.videoId AS youtubeVideoId,
    v.url AS videoUrl,
    v.title,
    v.description,
    v.thumbnail AS videoThumbnail,
    v.publishedAt,
    -- Datos del producto (puede ser NULL si no hay vinculación)
    pc.productId AS itemId,
    pc.sku AS itemSku,
    pc.productTitle AS itemTitle,
    -- URL del producto: https://libreopcion.com.ar/{titulo-con-guiones}-P{id}
    CASE 
        WHEN pc.productId IS NOT NULL THEN 
            'https://libreopcion.com.ar/' 
            + REPLACE(LOWER(pc.productTitle), ' ', '-') 
            + '-P' 
            + CAST(pc.productId AS VARCHAR)
        ELSE NULL 
    END AS itemUrl,
    pc.productImage AS itemImage,
    pc.listPrice AS itemListPrice,
    pc.discount AS itemDiscount,
    pc.finalPrice AS itemFinalPrice,
    pc.sellerId AS itemSellerId,
    pc.sellerName AS itemSellerName,
    CAST(pc.envioGratis AS BIT) AS itemFreeShipping,
    CAST(pc.instantFlash AS BIT) AS itemInstantFlash,
    -- Prioridad de la vinculación para ordenamiento
    vp.priority AS linkPriority
FROM LO.dbo.yt_videos v
LEFT JOIN LO.dbo.yt_video_products vp 
    ON vp.video_id = v.id
LEFT JOIN ProductosCompletos pc ON (
    -- Vinculación por item_id específico
    (vp.item_id IS NOT NULL AND pc.productId = vp.item_id)
    OR 
    -- Vinculación por id_interno (ya filtrado al más nuevo)
    (vp.internal_id IS NOT NULL AND pc.id_interno = vp.internal_id)
)
WHERE v.label = :label
  AND v.deleted = 0
  AND v.show = 1
ORDER BY v.createdAt DESC, vp.priority ASC
 
 
```
