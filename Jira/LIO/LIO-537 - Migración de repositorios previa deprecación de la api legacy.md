## Descripcion

Este epic agrupa las historias necesarias para migrar recursos desde la API legacy hacia la nueva API v4, reemplazando dependencias intermedias y unificando la lógica en una arquitectura más mantenible, escalable y consistente. El objetivo es asegurar compatibilidad funcional con los endpoints actuales, mejorar la performance y facilitar la evolución futura del sistema.
