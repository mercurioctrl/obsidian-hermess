## Descripción

Se agregará un nuevo objeto al payload en el PATCH y a el response en el GET para el caso de medio de pago id `5062` Tarjeta de debito y crédito

```
PATCH https://api.gamma.libreopcion.com/pedidos/checkout
```

