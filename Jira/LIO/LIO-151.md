## Descripción

Crearemos un ejemplo de procesamiento, funcional con la API v4 que nos permita implementar en el checkout la pasarela de pagos de mercado pago. 

[attachment]
- Ejemplo basico de integracion de mercadopago con vue js.



Documentación utilizada: 

[link](https://www.mercadopago.com.ar/developers/es/docs/checkout-api/integration-configuration/card/integrate-via-core-methods) 



**Recurso: **El objetivo del recurso es poder ser capaz de procesar un pago según el proveedor inicializado desde el front.

```
POST {{url}} /v4/payment/{proveedor_de_pago}/process
```

`proveedor_de_pago`: se debe pasar el proveedor que se quiere utilizar. 

- mercadopago


- payway


- gernet


- otros..





```
POST {{url}} /v4/payment/mercadopago/process
```
