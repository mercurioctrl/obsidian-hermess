## Descripción

Se debe crear una clase repositorio que retorne un listado único con todos los posibles estados que devuelve 

- Los estados deben ser los mismos que devuelve el método `PaymentStatusDto::determinePaymentStatus()`.


- No debe haber duplicados.


- El repositorio debe considerar distintos escenarios posibles combinando los parámetros como se realizo en [link](https://lioteam.atlassian.net/browse/LIO-298) 



```
GET {APIv4_URL}/api/paymentStatus
```

Retorna

```
[
  "Pago aprobado",
  "Pago Acreditado",
  "Pago aprobado, devuelto parcialemente",
  "Esperando Pago",
  "Verificando comprobante de pago",
  "Pendiente de pago"
]
```
