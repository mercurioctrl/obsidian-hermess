## Descripción

Retomando lo realizado en el hilo [https://api.nbe.com.ar/v1/priceListEfinderABB/4520348d5e5ab6d973b54aec33dcd3](https://api.nbe.com.ar/v1/priceListEfinderABB/4520348d5e5ab6d973b54aec33dcd3) se busca realizar una automatización para realizar una subida del archivo actualizado periódicamente.

Para esto crearemos un recurso similar al siguiente

```
POST {{API_URL}}/v1/syncUp/priceListEfinderABB?token={EfinderABBToken}
```

Que realice lo mismo que se realiza todos los días manualmente, que es lo siguiente:

1 - Se ingresa al sitio [link](https://www.netcomponents.com/) 

2 - Con las credenciales compartidas por slack, ingresamos mediante el login

[attachment]
3 - Nos dirigimos a la pestaña “upload” para subir el archivo generado por [https://api.nbe.com.ar/v1/priceListEfinderABB/4520348d5e5ab6d973b54aec33dcd3](https://api.nbe.com.ar/v1/priceListEfinderABB/4520348d5e5ab6d973b54aec33dcd3) 

[attachment]
