Hace mas de un año que informe de este problema, se vio con desarrollo de Qloud que es la empresa que nos provee del enlatado web y a la mayoría de los locales de nuestro tamaño, ellos revisaron, lanzaron un comunicado, yo volví a hablarlo con ustedes y hasta el día de hoy seguimos igual.

Nosotros casi el 90% de las compras que les hacemos son mediante las ventas de la api y la verdad que este último tiempo fueron muchos los problemas que nos generó, reitero nuevamente lo que pasa, todos los días a las 11am y 11pm sin excepción, la api de ustedes le comparte a nuestra web todos los productos sin excepción, es decir, no importa si hay stock o no, no importa el precio, no importa nada, en nuestra web aparece todo el catálogo de ustedes como si hubiera stock, y que pasa, la mayoría de los precios claramente están completamente desactualizados, la gente hace sus compras, a nosotros nos llega por ejemplo la venta de una 5070 a $995.000, o dos modulos de ram de 32GB DDR5 a $360.000, los clientes hacen los pagos, todo normal y eso nos genera una posterior discusión con el cliente y esta ultima vez hasta una demanda en COPREC, por ende desde la empresa tomamos la decisión lamentablemente de que si no pueden resolver este error a la brevedad, daremos de baja la api y con ello todas las ventas mensuales que realizamos, ya no es sostenible esta situación, ninguno de los otros proveedores que este último tiempo estuvimos dando de alta preparándonos para suplir todo lo que compramos con ustedes con ellos, tienen este problema, ya sea Elit o Air por ejemplo.
