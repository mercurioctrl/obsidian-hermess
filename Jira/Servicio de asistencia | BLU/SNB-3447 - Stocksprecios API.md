## Descripcion

Buen día,

Entre ayer y hoy tuve tres casos de clientes distintos que tuvieron errores con stocks o precios de la API (clientes les compraron productos que no tenemos hace tiempo o en su web figuran precios reflejados con márgenes mínimos).

Les paso algunos casos:

Cliente 34065 - BATTA LUCIANO EMANUEL: Le compraron por API esta semana MOTHER ASUS (AM5) PRIME B650M-A AX6 II y PLACA DE VIDEO GIGABYTE RADEON RX 7700 XT GAMING OC 12GB, ambos productos sin stock hace tiempo.

Cliente 80158 - VIDAL JUAN PABLO BENITO: Le compraron por API esta semana MONITOR GAMER GIGABYTE 26.5 AORUS FO27Q3 SA, producto sin stock hace tiempo.

Cliente 79896 - ARENA ALEJANDRO PABLO: Tiene productos publicados, como por ejemplo PROCESADOR AMD (AM4) RYZEN 5 5600GT (adjunto imagen), que los marcó al menos 6 puntos por API pero en su ecommerce solo figuran cargados con menos de 2 puntos de margen entre su costo y el precio que paga el usuario final.

Le avisé a los tres que les escriban al mail de integraciones para revisar cada caso puntual, pero igualmente cargo el reporte por las dudas que haya algún tema puntual ya que todo esto fue entre ayer y hoy.

Saludos.
