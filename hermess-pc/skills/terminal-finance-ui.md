# Skill: terminal-finance-ui

Aplica una estética de **dashboard financiero / terminal** a interfaces web.
Inspirado en bondterminal.com y gitReporter.

## Cuándo usarlo

- Replicar sistemas ERP o dashboards de datos con look "terminal"
- Interfaces con estética de trading / finanzas
- Proyectos que requieran apariencia densa y técnica (no consumer)

## Cómo invocarlo

```
/terminal-finance-ui
```

## Características visuales

- Fondo **dark navy** oscuro
- Fuentes **monospace**
- Sin gradientes ni blur
- Layout denso con **bordes finos**
- Alta densidad de información en pantalla
