# Skill: mini-saas-ui

Crea interfaces pixel-perfect para el **Mini SaaS de BLU** usando el design system exacto del proyecto.

## Cuándo usarlo

- Crear una nueva página, componente, modal, card o sección dentro del sistema Mini SaaS
- Replicar la UI de presupuestos, proyectos u otras secciones del sistema BLU
- Mantener consistencia visual con el design system del Mini SaaS

## Cómo invocarlo

```
/mini-saas-ui
```

## Características

- Genera componentes con el design system exacto del proyecto BLU
- Interfaces pixel-perfect que siguen las convenciones visuales establecidas
- Integra con el stack Nuxt 3 del frontend
