# Producto: Bot Copiloto Nativo para WhatsApp

## Contexto
Idea registrada a partir de una nota de voz de Catriel el 17 de Mayo de 2026.

## Descripción General
Un bot conversacional (similar a Bily) diseñado para operar de manera nativa y directa desde WhatsApp.

## Características Clave (Requerimientos)
1. **Lectura de Contexto Completo:** Capacidad de acceder, leer y analizar todo el historial y el contexto de los chats en WhatsApp.
2. **Interfaz Nativa:** Todo el enfoque e interacción se maneja directamente desde WhatsApp, sin interfaces externas.
3. **Integración con APIs:** El bot debe estar conectado a diversas APIs externas que le permitan ejecutar distintas acciones y herramientas especializadas.

## Notas Técnicas / Estado
- **Etapa:** Ideación / Conceptualización.
- **Core del producto:** Interfaz conversacional que comprende el contexto humano en WhatsApp y actúa sobre él delegando tareas a APIs externas.
