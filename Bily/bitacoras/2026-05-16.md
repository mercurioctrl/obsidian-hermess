# Bitácora Diaria - 2026-05-16

## Tareas Hechas
- **Obsidian / RRHH:** Se crearon los perfiles individuales de los miembros del equipo Blu en `Bily/personas/` a partir de las notas de RRHH.
- **Hogar / Reforma Lavadero:** Se actualizaron y organizaron los pagos recientes a Mario (contratista) en la nota correspondiente, totalizando $5.400.000.
- **Seguridad y Red Local:** Se usó a Claude Code para escanear la red buscando cámaras. Se detectaron equipos Dahua y una IP genérica (.192). Se comprobó mediante intentos de extracción de frames que las contraseñas por defecto han sido cambiadas (están seguras).
- **Cerebro Común:** Se le otorgó un espacio a Claude en la bóveda creando `Claude/MEMORIA.md` y documentando cómo usar el REST API local.
- **Identidad de Bily:** Se actualizó `IDENTITY.md` tras recibir un diseño de avatar oficial generado por IA (robot blanco, pantalla negra, ojos digitales amigables).
- **Desarrollo (avatarMinter):** Se inició y evolucionó un proyecto Node.js/Express para generar avatares SVG. Pasamos de formas abstractas a identicons 8-bits, y luego a Claymorphism 3D con IA (Pollinations), para finalmente volver a SVG puro.

## Decisiones
- **Orquestación de Claude Code:** Por limitaciones del entorno local (`SIGTERM` en sesiones interactivas largas), se acordó que las sesiones de Claude se abrirán manualmente en la terminal de Catriel y yo facilitaré los prompts optimizados.
- **Estilo de Avatares:** Ante los cortes por rate limit en APIs de imágenes, se decidió volver a un generador SVG puramente procedural, pero con calidad premium (fondo negro, glow CRT, colores neón) y basado en plantillas fijas simétricas.
