# MKT-314 · Sistema de avisos y notificaciones automáticas (restock + push)

[MKT-314 en Jira](https://bluinc.atlassian.net/browse/MKT-314)

## Descripción

## Descripción

Sistema de comunicación automática / event-driven que cubre dos subideas relacionadas:

## A. "Avisarme cuando llegue" (restock)

Para productos sin stock:

- El usuario se suscribe al producto
- Cuando vuelve el stock se dispara:  - Mail automático
  - Notificación

## B. Push notifications web / mobile

Notificaciones push del navegador / celular:

- El usuario acepta los permisos
- Luego recibe avisos sobre:  - Ingresos
  - Promos
  - Lanzamientos
  - Novedades

## Consideraciones

Evaluar una solución propia para no depender de plataformas caras tipo Brevo / Sendinblue.

## Tema (abstracto)

Comunicación automática / event-driven (alerts / push / restock).
