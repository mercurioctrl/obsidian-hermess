# COB-621 · API - Feat - Facturar Remitos / Órdenes de Compra

[COB-621 en Jira](https://bluinc.atlassian.net/browse/COB-621)

## Descripción

### Consultar remitos

```
GET /afipPurchases/orders[/{stringFilter}]
```

Query params: `currentPage`, `itemsPerPage`, `between` (`dd-mm-yyyy_dd-mm-yyyy`), `estado` (`facturado` | `sinFacturar`), `sucursal`

```json
{
  "response": [
    {
      "orderNumber": 1234,
      "date": "2026-04-21",
      "warehouseCode": "SAF",
      "currencyCode": "PSO",
      "exchangeRate": 1.0,
      "invoiced": false,
      "invoiceNumber": null,
      "supplier": "PROVEEDOR SA",
      "supplierCode": "PRO001",
      "netTotal": 15000.00
    }
  ],
  "pagination": { "total": 80, "current": 1, "pageSize": 15 }
}
```

> `invoiced: true` → el remito ya fue facturado. Ocultar/deshabilitar el botón "Completar".

### Generar facturas

```
POST /afipPurchases/orders/{orderId}/invoice
```

**Body** (campos requeridos: `dfecfac`, `ccodpago`, `ccoddiv`, `nvaldiv`, `ntipocomp`, `lines`):

```json
{
  "dfecfac": "2026-04-28",
  "dfeccont": null,
  "ccodpago": "CO",
  "cnumsucpag": "0001",
  "ccodalm": "SAF",
  "ccoddiv": "PSO",
  "nvaldiv": 1,
  "ntipocomp": 1,
  "observation": null,
  "lines": [
    { "cref": "ART001", "cdetalle": "Artículo", "npreunit": 7500, "ncanent": 2, "niva": 21, "ndto": 0 }
  ],
  "taxes": [
    { "ccodimp": "ICA", "cdescrip": "IIBB CABA", "nporctasa": 0.5, "nimpbase": 15000 }
  ]
}
```

> `cserie` y `csufac` son autogenerados por el backend. No enviar.  
> `dfeccont` es opcional — si es `null` toma el valor de `dfecfac`.  
> `cnumsucpag` y `ccodalm` son opcionales — defaults: `"0001"` y `"SAF"`.

**Response:**

```json
{ "cnumfac": 37520, "csufac": "A-0002-00037520" }
```
