# LIO-631 · API - Review - Migrar recurso para guardar en favoritos desde la ficha del producto -> Homologar respuesta al obtener los favoritos

[LIO-631 en Jira](https://bluinc.atlassian.net/browse/LIO-631)

## Descripción

El endpoint `GET /v4/favoritos` no devuelve la misma estructura que el endpoint legacy `GET /favoritos`.







Actualmente v4 responde con campos planos y valores como string, mientras que legacy devuelve una estructura agrupada y tipada, con objetos como:

- `estados`
- `precios`
- `cuotas`
- `envios`
- `marca`
- `vendedor`

Esto puede afectar al frontend, ya que la migración debería conservar el contrato esperado por el cliente.
