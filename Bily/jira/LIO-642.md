# LIO-642 · API task v3 feat push deeplink legacy

[LIO-642 en Jira](https://bluinc.atlassian.net/browse/LIO-642)

## Descripción

### Resumen

Actualmente, las notificaciones push enviadas a la app móvil del vendedor no incluyen la información necesaria para que la app sepa a qué pantalla navegar al tocarlas.

### Contexto

Se extiende el servicio de procesamiento de notificaciones push del flujo legacy.

No se requieren cambios en los servicios que generan las notificaciones.

### Criterios de aceptación

- Al momento de enviar la notificación al dispositivo, se adjunta automáticamente el contexto de navegación.
- El contexto de navegación incluye:  - tipo de notificación
  - identificadores del recurso involucrado
- El contexto se infiere a partir de los datos que ya existen en el sistema.
