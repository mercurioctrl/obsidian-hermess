# SNB-3582 · Restringir duplicados en pedclit por idLo

[SNB-3582 en Jira](https://bluinc.atlassian.net/browse/SNB-3582)

## Descripción

```
SELECT idLo, COUNT(*) AS cantidad
FROM pedclit
WHERE idLo IS NOT NULL
GROUP BY idLo
HAVING COUNT(*) > 1;
```
