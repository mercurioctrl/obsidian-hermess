# MKT-313 · Popup promocional administrable

[MKT-313 en Jira](https://bluinc.atlassian.net/browse/MKT-313)

## Descripción

## Descripción

Popup que aparece una vez por día, al primer login / entrada del usuario, con promociones o ingresos relevantes.

## Características

- Configurable desde CMS
- Activable / desactivable
- Editable sin depender de desarrollo
- Pensado como soporte visual de campañas de WhatsApp / mail / redes
- Frecuencia: 1 vez por día, primer ingreso del usuario

## Ejemplos de uso

- "Ingresó X procesador"
- "Promo Thermaltake"
- "NB Travel"
- Etc.

## Tema (abstracto)

Comunicación contextual invasiva-controlada (popup).
